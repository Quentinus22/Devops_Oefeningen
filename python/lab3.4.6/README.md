Project: Python Objectgeoriënteerd Programmeren (OOP)
In dit deel van het project wordt de overstap gemaakt van script-gebaseerd programmeren naar Objectgeoriënteerd Programmeren. Er is een herbruikbare blauwdruk (Class) ontwikkeld voor het beheren van locatiegegevens.

1. Class Definitie & Structuur (Location)
Er is een Python-klasse gedefinieerd om data en functionaliteit te groeperen:

Initialisatie: De __init__ methode wordt gebruikt om bij de aanmaak van een object direct de eigenschappen name (naam) en country (land) vast te leggen.

Methodes: De methode myLocation() is toegevoegd om de opgeslagen data op een geformatteerde manier naar de console te printen.

2. Instanties & Objectbeheer
Om de werking van de klasse te demonstreren, zijn er meerdere unieke objecten (instanties) aangemaakt:

Testdata: Er zijn drie standaard locaties aangemaakt voor testpersonen uit Portugal, China en Kenia.

Persoonlijke Implementatie: Er is een specifieke instantie your_loc aangemaakt voor de beheerder (Glineur Quentin) met de waarde "België".

3. Uitvoering & Output
Door de methode .myLocation() aan te roepen voor elk object, wordt bewezen dat de logica correct werkt voor verschillende datasets binnen dezelfde klasse-structuur.
